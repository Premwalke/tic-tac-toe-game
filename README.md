
Mini Game Project – Tic Tac Toe


Aim:

To create a console-based mini game in C++ that demonstrates core programming concepts such as loops, arrays, and conditional logic.
Objective
To understand game development basics using C++.
To implement loops, arrays, and conditional statements.
To create an interactive two-player Tic Tac Toe game.
To implement win/loss detection and replay functionality.



Software Requirements:

Programming Language: C++
Compiler: Turbo C++ / CodeBlocks / VS Code
Operating System: Windows/Linux



Concepts Used:

Arrays
Loops
Functions
Conditional Statements
Switch Case
Object-Oriented Programming (optional)


Key Features:

Dynamic game board display
Two-player gameplay
Win and draw detection
Input validation
Replay option
Console-based interactive interface
Game Rules
The game is played between two players.
Player 1 uses X and Player 2 uses O.
Players take turns choosing positions from 1 to 9.
The player who forms a row, column, or diagonal first wins.
If all positions are filled without a winner, the game ends in a draw.


Algorithm:
Step 1: Start the program
Step 2: Initialize the game board
Step 3: Display the board
Step 4: Ask current player for position
Step 5: Update board if position is valid
Step 6: Check for winner or draw
Step 7: Switch player and continue
Step 8: Ask for replay option
Step 9: Stop the program
